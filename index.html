Multiple Term Uploader

Use this page to upload terms from a .csv file. The file must contain the following columns, in the order below:

  1. Term
  2. Acronym
  3. Description

To upload terms

  1. Use the Choose Files button to select a csv file.
  2. Use the Template list to apply a metadata template to each term.
  3. Click Upload to upload the file's content to the Corporate Dictionary

Note

When you upload the list, if any terms in your list already exist in the dictionary, the existing terms will be overwritten.
